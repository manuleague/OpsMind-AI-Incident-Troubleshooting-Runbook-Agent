# Answer Formatting Prompt

Format the answer with these sections:

1. Incident Summary
2. Grounded Evidence
3. Likely Diagnosis
4. Recommended Remediation
5. Validation Checks
6. Escalation Triggers
7. Requires Human Review
8. Citations

Requirements:

- Use bullets.
- Keep steps short and operational.
- Include source citations next to evidence-backed claims.
- Label confidence as low, medium, or high.
- Label risk as low, medium, or high.
- If evidence is insufficient, say so and ask for missing facts.
